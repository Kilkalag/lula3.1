Теоретическая справка

Уравнения высших степеней (степени 3 и выше) решаются различными методами, включая разложение на множители, использование специальных формул и теорем.

Формулы сокращенного умножения

Основные формулы:

  • (a + b)² = a² + 2ab + b²
  • (a - b)² = a² - 2ab + b²
  • (a + b)(a - b) = a² - b²
  • (a + b)³ = a³ + 3a²b + 3ab² + b³
  • (a - b)³ = a³ - 3a²b + 3ab² - b³

Метод группировки

Алгоритм решения:

  1. Сгруппировать слагаемые с общими множителями
  2. Вынести общий множитель за скобки в каждой группе
  3. Получить одинаковые выражения в скобках
  4. Вынести общую скобку как множитель

Схема Горнера

Алгоритм деления многочлена P(x) на (x - c):

  1. Записать коэффициенты многочлена
  2. Подставить значение c в схему
  3. Выполнить вычисления по схеме
  4. Последнее число - остаток от деления
  5. Если остаток 0, то c - корень многочлена

Теорема Безу

Формулировка:

Остаток от деления многочлена P(x) на двучлен (x - c) равен P(c).

Следствие:

Число c является корнем многочлена P(x) тогда и только тогда, когда P(x) делится на (x - c) без остатка.

Замена переменной

Методика:

  1. Выделить повторяющееся выражение
  2. Ввести новую переменную t = выражение
  3. Решить полученное уравнение относительно t
  4. Вернуться к исходной переменной

Биквадратные уравнения

Общий вид:

ax⁴ + bx² + c = 0

Метод решения:

  1. Сделать замену t = x²
  2. Решить квадратное уравнение at² + bt + c = 0
  3. Для каждого положительного корня t найти x = ±√t

Пример решения уравнения

Уравнение:

x³ - 6x² + 11x - 6 = 0

Ваше решение:

Пошаговое решение:

  1. Проверим x=1: 1 - 6 + 11 - 6 = 0 ⇒ x=1 корень
  2. Разделим многочлен на (x-1) по схеме Горнера
  3. Получаем: (x-1)(x² - 5x + 6) = 0
  4. Решаем квадратное уравнение: x² - 5x + 6 = 0
  5. Корни: x=2 и x=3
  6. Ответ: x ∈ {1, 2, 3}

Попробуйте подставить x=1 в уравнение. Если результат равен 0, то это корень.
