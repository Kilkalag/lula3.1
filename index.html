Специальный случай уравнений четвертой степени.

Пример решения уравнения

Уравнение:

x³ - 6x² + 11x - 6 = 0

Ваше решение:

Пошаговое решение:

  1. Проверим x=1: 1 - 6 + 11 - 6 = 0 ⇒ x=1 корень
  2. Разделим многочлен на (x-1) по схеме Горнера
  3. Получаем: (x-1)(x² - 5x + 6) = 0
  4. Решаем квадратное уравнение: x² - 5x + 6 = 0
  5. Корни: x=2 и x=3
  6. Ответ: x ∈ {1, 2, 3}
Правильно! Вы нашли все корни.
